- **Productivity metric**: Lines of code saved, $50/hr assumed rate

## Strategy

- Remove roadblocks (budget, IT, legal), enable training, surface best use cases
- Top-down mandate alongside bottom-up enablement — training alone won't drive it
- "Gemba walk": find who's using it and why, then who isn't and why not
- Escalation: if 20%+ still refuse after reasonable steps, escalate to DLT
- Dashboard to be shared with OpCo champions and heads of R&D
- Need tooltips/methodology context on dashboard to prevent derailing conversations
- Julie emailed OpCo presidents: "if someone isn't using the tools, assume they're less productive"
