## Summary

Detailed Cortex Code demo + Horizon Catalog deep dive. Cortex Code outperforms Tableau for speed-to-value (~4 hours to build finance demo). Horizon Catalog provides AI-generated table/column descriptions, role-based access, masking policies, data quality monitoring.

## Key Capabilities

- Natural language data interaction for non-technical users
- AI-generated metadata descriptions (overridable manually)
- Column/row-level security + masking policies (e.g., HR-only salary data)
- External data integration with lineage tracking
- Separation of storage and compute layers

## Architecture

- Open format investments (Iceberg support)
- Native catalog for connected data; external systems need Collibra or similar
- Custom ROI analysis available through value engineering team
