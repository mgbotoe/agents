# Microsoft Copilot Licensing & Feature Walkthrough

Granola meeting: `c12c67b4-4c8b-42c0-99fb-dbe04b084855`

## Summary

Breakdown of Copilot Chat (free) vs M365 Copilot (licensed). Key differentiator: licensed version includes "WorkIQ" — full org data access (SharePoint, OneDrive, Teams, meetings).

## Key Distinctions

- **Free**: Web search only + Outlook. No SharePoint/OneDrive/Teams access.
- **Licensed**: Full org data access, all M365 apps, Work/Web toggle visible in UI.
- **Agents**: Licensed users get free access to all org agents. Unlicensed pay $0.01/query.
- **License check**: Look for Work/Web toggle at top or "Premium"/"Basic" in footer.
