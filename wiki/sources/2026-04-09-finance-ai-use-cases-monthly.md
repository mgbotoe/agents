# Monthly Finance AI Use Cases — Martin, Chris Bouda, Nick Johnson

Monthly sync between EAI team and Finance stakeholders (Chris Bouda, Nick Johnson) to review and prioritize finance AI use cases.

## Use Case Prioritization Discussion

### Core Question
- Chris Bouda to EAI: help us categorize use cases — which are Copilot-level (simple), which truly need custom AI, and which are really IT/data infrastructure problems?
- Goal: keep EAI focused on "winner type things" that only AI can solve
- Madina had previously sent an email summary of all received use cases

### OpCo Sizing Strategy
- SciEx and IDBS are small → limited ROI potential
- Beckman and Cytiva are large → better targets for finance AI pilots
- Should prioritize use cases at the largest opcos first for maximum impact

### Data Infrastructure Bottleneck
- Different opcos on different ERPs (Oracle, SAP, etc.) → hard to build cross-opco AI tools
- Nick and Martin to meet separately on building the business case for Snowflake data lake mapping
- If finance data is mapped to a common structure in Snowflake → force multiplier: solutions built for one opco can be replicated across others
- Short-term: pilot use cases in one opco with direct ERP access
- Long-term: Snowflake data lake enables cross-opco scaling

### Use Cases Under Discussion
- Headcount forecasting — might be better solved in Workday than custom AI
- "Big heavy data" use cases — need Snowflake mapping first
- Martin to connect with John and add comments to the use case file

### Process Going Forward
- Keep meeting monthly to set priorities
- Madina to continue email summaries of incoming use cases
- Nick to provide SME guidance on which finance challenges are biggest
