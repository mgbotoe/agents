# Microsoft Copilot Studio — IP Intelligence Agent Prototype

Granola meeting: `a882db13-a0e0-4aa0-8762-1ae0b9a78c82`

## Summary

Microsoft-Danaher collaboration on patent analysis automation. Multi-agent system in Copilot Studio (GPT + Claude) processes 3,000–9,000 patents down to manageable subset. MVP complete after 10-week engagement.

## Key Capabilities

- **Relevance screening**: Semantic claims analysis, not keyword matching
- **Confidence scoring**: 0–1 scale per patent, analysts prioritize low-confidence for QC
- **Taxonomy categorization**: Analyst-defined logic + semantic suggestions
- **Report generation**: Standardized Excel templates with charts
- **Grounded knowledge**: SharePoint PDFs + Google Scholar prevent hallucination
- Human-in-the-loop at every phase

## Business Impact

- Analysis time reduction: 70–80% target
- Project throughput: from 2/analyst/year to 4–6/analyst/year
- Screening: ~5 weeks → ~1–1.5 weeks
- Manual QC effort: ~50% reduction
