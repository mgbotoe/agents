# Career-Ops

AI-powered job search automation system. Forked from santifer/career-ops.

## Workspace
`C:\Workspace\career-ops\`

## Technical Details
- **Stack:** Node.js (mjs modules) + Bash orchestration + Playwright (browser automation, PDF generation)
- **Dashboard:** Go TUI (Bubbletea, Catppuccin Mocha theme)
- **CV template:** HTML/CSS (Space Grotesk + DM Sans)
- **Data:** Markdown + TSV (applications, pipeline, scan history), YAML (profile, portals)
- **Mode system:** 14 modes (evaluation, scan, batch, apply, PDF, tracker, deep research, outreach, pipeline, interview prep)
- **Evaluation:** 6-block framework (A-F) with 10 weighted scoring dimensions
- **Portal scanner:** 45+ configured companies, 3-tier strategy (Playwright → API → WebSearch)
- **Data contract:** User layer (protected) vs System layer (auto-updatable)
- **Usage:** 11 evaluations, 110+ portal scans, 8+ STAR+R stories
