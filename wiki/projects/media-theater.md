# Media Theater (CineVault)

AI-powered media discovery app — browse, search, and get mood-based recommendations for movies and TV.

## Workspace
`C:\Workspace\Personal Projects\media-theater\`

## Technical Details
- **Stack:** Next.js 16 (Turbopack), React 19, TypeScript (strict), Tailwind CSS v4, Biome
- **Database:** Turso (SQLite edge) via Drizzle ORM
- **AI:** Claude API for mood-based recommendations
- **APIs:** TMDB + MDBList for media data
- **Testing:** Playwright E2E
- **Key patterns:** App Router, dual themes (Matinee/Night Screening), @theme inline design tokens, auth hardening, input validation
- **Recent work:** Test additions, auth hardening, input validation fixes
